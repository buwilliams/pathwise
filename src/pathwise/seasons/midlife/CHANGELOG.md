# Midlife — Revision History

## 0.1.0

Initial revision. Same formal model as build-independence v0.5.0
(``L = {V, T, A, K, W}``, viable/desirable filters, path-of-stages
scoring, per-decision recoverability) with a midlife-specific
reinterpretation:

- The conjecture is reframed: time is scarcer than at 18, but
  considerable momentum has already been built — the work is to
  *redirect* it. Emotions are the signal that current alignment has
  drifted, and creativity plus the courage to make real changes are
  the load-bearing inputs midlife life-design needs.
- Questionnaire reshaped for midlife concerns: relationship status,
  caregiving load, current career feeling, divorce/separation
  history, debt pressure, last meaningful change, readiness for
  change, biggest perceived block. Drops teen-only fields
  (``lives_with_parents``, ``has_car``, ``move_out_*``).
- Wants surface midlife options: career change, start a business,
  return to school, geographic move, growing the family or adopting,
  repairing or leaving a partnership, creative practice, sabbatical,
  simplifying.
- Path buckets renamed for the season: ``bold_change``,
  ``compounding_change``, ``creative_reinvention``. Scenarios cover
  career pivots, side-business build-out, sabbatical reset,
  geographic move, returning to school, and stay-and-repair.
